Molecule video presentation

  • Part 1: Datomic data model
  • Part 2: Building a DSL with Scala macros
  • Part 3: Tour of Molecule features
  • Part 4: Molecule domain modelling

Molecule presentation in 4 parts by creator Marc Grue. The talk took place at a Scala group MeetUp in Gothenburg, Sweden on April 25 of 2017 and is based on Molecule, version 0.10.3. Each part of the speech covers its own topics that are listed below each of the 4 videos.

Some changes has been made to the slides where the API was improved based on great input from the attendees:

  • All database filter getters follow the same verb form:
    • getAsOf(t)
    • getSince(t)
    • getWith(txData)
    • getHistory
  • All molecule gets now return type Iterable[(A, B, ...)] so that you can retrieve large data sets successively with an iterator.
  • Some unclarity with transactional meta data (like operations) on history data is being reviewed.
